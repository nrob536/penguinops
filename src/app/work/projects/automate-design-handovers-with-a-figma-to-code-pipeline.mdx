## Overview

In this project, I DO NOT intend to replicate the official StatsNZ clock but to demonstrate:


   - API design in both R and Python
   - Use of statistical standards (SDMX-style JSON output)
   - Deployment workflows with Docker and Fly.io
   - Performance comparison between R and Python implementations


## Disclaimer

The data and calculations are illustrative only and do not reflect real population data. This project is not affiliated with StatsNZ and is not meant to be used as an official population clock.
